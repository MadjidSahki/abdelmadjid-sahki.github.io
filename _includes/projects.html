Digger

Tehnologies

  • C# / ASP.NET Core / Bash
  • Dgraph - SQL
  • Vue.js - D3.js

Digger est un projet permettant de tracker et centraliser les données publiques que nous pouvons laisser derrière nous sur le web. A l’aide de différents outils intégrés vous allez pouvoir consulter toutes les traces que vous avez pu égarer sur votre chemin. Bien souvent compliqué à mettre en place, notre interface permettra à l’utilisateur final de faciliter l’installation de la multitude de logiciels d’OSINT.

ICAR

Tehnologies

  • C# / ASP.NET Core / Python / Java
  • SQL

ICAR est un proof of concept, visant à montrer la possiblité pour un robot, de se déplacer de manière autonome en intérieur, et se localisant simplement avec des photos du plafond. Pour cela, le robot effectuait un SLAM et avec de l'analyse poussée d'image, il pouvait faire une cartographie à l'aide des images du plafond.
